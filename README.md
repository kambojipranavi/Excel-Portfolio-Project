# Excel-Portfolio-Project
This project demonstrates advanced Excel skills using real business data, including:
- Data cleaning and transformation
- Pivot table analysis for insights
- A dynamic dashboard visualizing key metrics

* Features
- Data Sheet: Source data used throughout the project
- Pivot Sheet: Summarized reporting and ad hoc analysis
- Dashboard Sheet: Interactive, visual representation of key trends

* How to Use
Open Excel-portfolio-project.xlsx and start with the Dashboard sheet. Use slicers/filters to explore different data dimensions.

Skills Demonstrated: Data Analysis, Dashboarding, Excel Formulas, Business Intelligence
